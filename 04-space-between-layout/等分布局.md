# 等分布局

> 一行分为若干列，每一列的宽度相同

## 1. float

```css
float: left;
width: [100%/count]

/*父元素清除浮动*/
.clearfix:: after {
  content: '';
  display: block;
  clear: both; 
}
```

## 2. table

```css
/*父元素*/
display: table;

/*子元素*/
display: table-cell;
```

## 3. flex

````css
/*父元素*/
display: flex;

/*子元素*/
flex: 1;
````
